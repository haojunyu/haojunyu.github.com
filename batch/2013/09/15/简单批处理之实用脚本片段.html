    1. 用测试数据检验程序正确与否

:again
rem r.exe生成测试案例
r.exe > input
rem a.exe生成结果output.a
a.exe < input > output.a
rem b.exe生成结果output.b
b.exe < input > output.b
rem 检验output.a和output.b是否一致
fc output.a output.b > null
